## 주요 기능
○ 계정관리: 회원가입, 프로필 <br/>
○ 거래관리: 음악 구매, 음악 판매 <br/>
○ 마이페이지: 구매한 음악 목록 조회, 업로드한 음악 목록 조회, 프로필 수정 <br/>
○ 검색기능: 제목, 가수, 장르로 검색 <br/>
○ 차트기능: 노래 최신 차트, 조회순 차트 <br/>
